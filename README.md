## 프로젝트 명 
영화예약 사이트 구현 프로젝트


## 팀명
테라박스

## 구현기간
2023.12.04 ~ 2024.01.30(2개월)

## 개발목표
-	기존 레거시 사이트의 주요기능을 구현하여 MVC2 패턴을 이해한다.
-	기존 레거시 사이트의 비즈니스 로직을 이해한다.
-	CRUD 화면을 구현하여 기능의 흐름을 이해한다.
-	화면부터 Controller, DB 까지 전반적으로 구현한다.
